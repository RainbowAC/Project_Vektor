# Project Vektor

一个使用Godot 4.6和C#开发的2D游戏项目。

## 概述

Project Vektor是一个基于Godot游戏引擎和C#脚本的2D角色移动原型，展示了基本的玩家控制和移动机制。

## 技术细节

- **引擎**: Godot 4.6
- **编程语言**: C#

## 项目结构

```
Project_Vektor/
├── Assets/                 # 游戏资源（精灵图、纹理）
│   ├── test_sprite_blue.png
│   └── test_sprite_red.png
├── Scenes/                 # 场景文件
│   └── Characters/
│       ├── test_player.tscn
│       └── test_world.tscn
├── Scripts/                # C#脚本
│   └── Characters/
│       └── TestPlayer.cs
├── project.godot           # Godot项目配置文件
├── project_vektor.csproj   # C#项目文件
└── project_vektor.sln      # Visual Studio解决方案文件
```
